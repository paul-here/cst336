Let's dive right in! PHP stands for "Personal Home Page" and was developed by Rasmus Lerdorf in 1994. In 1995, the source code was released to the public as Personal Home Page tools. The next important milestone for PHP was the 1997 release of PHP 3. This release's inclusion of native support for mySQL helped propel PHP past the major competition: Perl.

Now, PHP is used by approximately 80% of all websites!
