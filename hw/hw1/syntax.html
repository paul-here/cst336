Intro Basic Syntax Dev Environments Resources

PHP can be inserted directly into an html file with the php tags. This done with the php tags as follows: <&php #PHP code goes here &>

Their are two types of comments:

  • Line comments use '#'
  • Multiline comments use '/* */'

Case sensitivity:

  • Keywords are not case sensitive
  • Variables are case sensitive

Finally, note that all PHP statements end with a ';'
